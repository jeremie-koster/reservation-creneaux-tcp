## Démarrer

### En local

Pour exécuter le code en local, veiller à fournir un fichier `credentials.yml` contenant vos identifiants de connexion au site web du TCP, de la forme suivante :

```yml
- user: <nom d'utilisateur>
- password: <mot de passe>
```

## Contribuer
